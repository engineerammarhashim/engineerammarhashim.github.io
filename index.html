Projects

SDR-Based Experiments for Future 6G Frequencies

Conducted SDR-based wireless communication experiments using GNU Radio Companion and HackRF, including transmission and reception of text, image, and audio files at higher frequency ranges.

Satellite Constellation Design for Optimal Coverage

Designed satellite coverage solutions for Cardiff, Perth, and Durban across S, UHF, and VHF bands, including link budget analysis and weather impact studies.

Design of an 8-Bit Configurable Timer Peripheral

Designed a configurable timer peripheral with auto-reload register functionality and two operating modes: one-shot and repeat mode, using digital design principles.

FPGA-Based Electrical Energy Monitoring System

Developed a Verilog-based FPGA energy monitoring system using voltage and current sensors, GSM SIM900A communication, and IoT/cloud data transmission. Compared cost and performance with an Arduino-Bluetooth implementation.

Hydrogen Fuel Cell with a Boost Converter

Simulated a hydrogen fuel cell power system in Simulink, stepping voltage from 40V to 70V using a boost converter. Replaced it with a Ćuk converter and compared output waveform behaviour using the same R, L, and C values.

Dynamic Wireless Charging System for Electric Vehicles

Designed a 2 kW Level-1 wireless EV charging system for a 200V peak battery at 10A and 50 kHz, analysing inductive power transfer, coil behaviour, and charging waveforms.

Methods for Analyzing Discrete-Time Systems

Analysed discrete-time systems using time-domain convolution, DFT-based frequency analysis, and Z-transform methods to evaluate output sequences, stability, and system response.

Spectrum Analysis of LED and Tungsten Using Spectrophotometer

Performed spectrum analysis of LED and tungsten light sources using PASCO spectrophotometer equipment, optical sensor alignment, and graphical data analysis through PASCO software.
